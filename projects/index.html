SlayTix

SlayTix is a movie ticketing web application that allows users to browse currently showing and upcoming movies, watch trailers, and search titles dynamically. The platform integrates a Spring Boot and MySQL backend with a React/Next.js frontend to manage movie data, implement search and filtering functionality, and support a booking flow for showtimes and tickets.

UGA Event Hub

UGA Event Hub is a web app that lets verified UGA students create, share, and RSVP to campus events. Built with Next.js, Express, and MongoDB, it features event filtering, profile pages, image uploads, and address metadata via the OpenCage API.

Royal Flush

Royal Flush is a 2D point-and-click card game developed in Unity that blends narrative and strategy elements. The project involved designing core gameplay systems, implementing a state-based UI flow, and integrating custom art and assets to create a cohesive and immersive experience.
